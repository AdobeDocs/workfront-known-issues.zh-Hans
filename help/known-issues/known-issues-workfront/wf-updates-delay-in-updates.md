# 更新：更新延迟

>[!NOTE]
>
>此问题已于 3 月 27 日修复。2025年。

当应该出现在对象更新流中的更新发生时，它最多在20分钟内不会出现。 更新已记录并可通过API请求找到，但它不会立即显示在更新流中。
