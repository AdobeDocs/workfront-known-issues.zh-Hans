# 验证：无法从直接链接打开验证

>[!NOTE]
>
>此问题已于2025年6月23日修复。

当用户尝试从直接链接打开验证时，验证未打开。 即使用户正在验证的工作流中，或者用户是Workfront管理员，也可能发生此情况。

**解决方法**

转到Workfront中的验证，然后选择“打开验证”选项。
