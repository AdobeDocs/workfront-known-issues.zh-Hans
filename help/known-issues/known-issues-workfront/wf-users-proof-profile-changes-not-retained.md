# 用户：验证配置文件的更改问题

>[!NOTE]
>
>此问题已于2025年12月17日修复。

已报告以下有关对用户的验证配置文件进行更改的问题：

* 当用户将不同的验证配置文件分配给用户并进行保存时，更改未保存，并且用户保留以前的验证配置文件。
* 用户的验证配置文件可能显示为不正确的验证配置文件，即使分配了正确的验证配置文件也是如此。
* 对用户进行其他更改时，他们的验证权限也可能更改。

_首次报告于2025年11月24日。_
