# 自定义Forms:将请求转换为项目时，不会保留按节中断隐藏的字段中的数据

>[!NOTE]
>
> 此问题已于2022年8月11日修复。

当用户将请求转换为项目，并且请求包含一个自定义表单，该表单的节点中包含数据（可以使用显示逻辑隐藏），则该节中的数据不会传递到新项目。

_第一次报告是在2022年7月27日。_
