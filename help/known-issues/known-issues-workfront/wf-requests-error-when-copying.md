# 请求：复制请求时出现“您必须是系统管理员”错误

>[!NOTE]
>
>此问题已于2026年2月3日修复。

当用户复制请求并尝试保存时，请求无法保存，且用户看到以下错误消息：

“您必须是系统管理员才能更改此自定义数据参数值。”

_首次报告于2025年1月12日。_
