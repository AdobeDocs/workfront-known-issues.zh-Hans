# 任务：创建任务时未保存计划的小时数

>[!NOTE]
>
>此问题已于2025年12月3日修复。

当用户创建任务并输入规划小时数，然后保存任务时，未保存规划小时数。

**解决方法**

创建任务，然后编辑该任务以添加计划小时数。

_首次报告于2025年10月10日。_
