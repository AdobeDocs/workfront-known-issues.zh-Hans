# 组：子组显示不正确

当用户在“设置”区域查看组列表时，他们会看到子组未正确列在父组下方。 子组正确地保存在父组下，但列表可能会导致混淆。

如果用户打开子组，他们可以在痕迹导航中看到子组已正确保存在父组下。
