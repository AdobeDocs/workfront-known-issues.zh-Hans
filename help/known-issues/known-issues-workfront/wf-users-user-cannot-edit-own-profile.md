# 用户：用户无法编辑其自己的用户设置

当具有 [!UICONTROL 工作], [!UICONTROL 审阅]或 [!UICONTROL 请求] 尝试编辑其自己的用户设置，打开的弹出窗口为空，用户无法进行任何编辑。 要退出弹出窗口，用户必须刷新页面。

**解决方法**

具有 [!UICONTROL 计划] 许可证可以为用户进行更改。
