# 任务：在任务分配中搜索名字不会返回用户

当用户进行任务分配并尝试按名字搜索被分配人时，搜索不会返回任何用户，并且用户会看到消息“未找到结果”。

**解决方法**

按全名搜索将按预期返回用户。
