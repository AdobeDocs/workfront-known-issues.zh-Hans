# 任务：在任务分配中搜索名字无法返回用户

>[!NOTE]
>
>由于无法重现此问题，因此已关闭它。

当用户进行任务分配并尝试按名字搜索受让人时，搜索不会返回任何用户，并且用户会看到“未找到结果”消息。

**解决方法**

通过全名搜索会按预期返回用户。
