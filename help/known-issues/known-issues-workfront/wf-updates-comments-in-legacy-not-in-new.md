# 更新：在旧版评论Experience中所做的评论在新版Experience中不可见

>[!NOTE]
>
>此问题已于2024年1月25日修复。

当用户在旧版评论体验中发表评论，并且该评论显示在旧版体验的评论区域时，相同的评论可能不会显示在新评论体验中。 这可能会导致使用新体验的用户错过评论。


_首次报告于2024年1月23日。_
