# 集成：通过集成发送到 AEM 时文件名为“空”

当通过 Workfront 集成将大文件（超过 100 MB）发送到 Adobe Experience Manager 时，AEM 中的文件名为&quot;空&quot;。

ZIP 和 TIF 文件均已报告此问题。

**解决方法**

执行下列操作之一：

* 将文档名称映射到 AEM 标题。
* 直接在 AEM 中输入文件名。
