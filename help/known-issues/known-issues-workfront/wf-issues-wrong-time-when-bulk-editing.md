# 问题：在批量编辑问题时，时间字段保存的时间有误

当用户批量编辑问题，然后为日期字段选择日期和时间并保存时，在问题中保存到此字段的时间不是用户选择的时间。而是在用户保存时似乎将该时间转换为 UTC。

已在报告和列表中都报告过此问题。
