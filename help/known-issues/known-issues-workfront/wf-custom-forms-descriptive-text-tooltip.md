# 自定义表单：描述性文本字段显示工具提示

>[!NOTE]
>
>此问题已于2022年9月13日修复。

当用户查看具有描述性文本字段的自定义表单时，描述性文本字段会显示工具提示。 如果用户将鼠标悬停在工具提示上，则工具提示会显示与描述性文本字段相同的文本。

_第一次报告是在2022年8月17日。_
