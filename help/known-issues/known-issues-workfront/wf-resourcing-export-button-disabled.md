# 资源规划者：“导出”按钮被禁用

>[!NOTE]
>
>此问题已于2025年6月9日修复。

当用户尝试从资源规划者导出时，按钮被禁用（灰显）并且用户无法导出。

_首次报告于2025年5月20日。_
