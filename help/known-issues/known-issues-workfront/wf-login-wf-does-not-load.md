# 登录：登录后不加载Workfront

>[!NOTE]
>
>此问题已于2026年2月19日修复。

当用户登录到Workfront时，Workfront不加载。 如果用户刷新页面，则Workfront会按预期加载。

**解决方法**

刷新页面。
