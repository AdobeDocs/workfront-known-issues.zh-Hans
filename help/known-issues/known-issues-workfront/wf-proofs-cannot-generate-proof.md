# 验证：无法生成验证

当用户尝试生成验证时，验证未创建，并且用户看到以下错误消息：

&quot;[!UICONTROL 生成校对时出现错误]&quot;

当用户的访问级别设置时，会发生这种情况[!UICONTROL  查看联系信息] 设置为“禁用”。

**解决方法**

添加要从中创建验证的文档 [!DNL Workfront]，然后从中创建验证 [!UICONTROL Workfront].
