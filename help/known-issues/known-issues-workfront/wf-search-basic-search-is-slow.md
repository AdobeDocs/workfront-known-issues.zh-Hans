# 搜索： [!UICONTROL 搜索] 极慢

当用户使用基本搜索( [!UICONTROL 搜索] 图标)，则返回搜索结果的速度会非常慢。

**解决方法**

使用[!UICONTROL 高级搜索]。指向的链接 [!UICONTROL 高级搜索] 位于下拉列表的底部，此下拉列表是在您单击 [!UICONTROL 搜索] 图标。
