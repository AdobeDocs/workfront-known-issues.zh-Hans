# 布局模板：创建布局模板时无法选择组

>[!NOTE]
>
>产品团队目前正在评估此问题的解决方案，这可能需要产品增强功能。 产品增强功能在“产品公告”中而非“维护更新”中传送。

当作为组管理员的用户尝试在[!UICONTROL 设置]的[!UICONTROL 界面]区域，提示用户在[!UICONTROL 管理权限]字段选择组。 用户键入组名，但该组未出现在下拉列表中且无法选择。 因此，用户无法保存布局模板。

**解决方法**

在“设置”的[!UICONTROL 组]区域，从组页面创建布局模板。
