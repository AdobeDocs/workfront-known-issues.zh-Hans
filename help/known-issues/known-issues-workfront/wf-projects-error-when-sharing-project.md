# 项目：共享项目时出错

当用户尝试共享项目时，项目不会共享，并且用户会看到以下消息：

“您没有足够的权限删除此项目”

即使用户尝试共享而不是删除项目，也可能发生这种情况。

_首次报告于2024年4月25日。_
