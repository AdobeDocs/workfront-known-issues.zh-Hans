# 文档集成：将资产拖到Experience Manager Assets集成中的文件夹时出错

当用户尝试将资产拖到通过Experience Manager Assets集成链接的文件夹时，文档未移动，并且用户看到以下错误消息：

“？document.action.upload.drag.noaccess？”

**解决方法**

使用“新增”按钮将资源添加到文件夹。

_首次报告于 2023 年 9 月 18 日。_
