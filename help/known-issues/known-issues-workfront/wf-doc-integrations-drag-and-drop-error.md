# 文档集成：将资源拖入 Experience Manager Assets 集成中的文件夹时出错

当用户尝试将资源拖至通过 Experience Manager Assets 集成链接的文件夹时无法移动文档，并且用户看到以下错误。

&quot;?document.action.upload.drag.noaccess?&quot;

**解决方法**

使用“新增”按钮将该资源添加到文件夹。
