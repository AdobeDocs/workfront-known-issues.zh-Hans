# [!UICONTROL 主页]: [!UICONTROL 主页] 使用 [!DNL Chrome]

>[!NOTE]
>
>此问题已关闭，因为无法重现。


当用户查看 [!UICONTROL 主页] 页面，并且使用的浏览器不是 [!DNL Chrome]，则他们可能会看到以下问题：

* 用户无法编辑字段，也无法向空字段添加值。
* 即使显示的工作对象上不存在字段，字段仍可能显示。

以下浏览器报告了这些问题：

* [!DNL Safari]
* [!DNL Firefox]
* [!DNL Microsoft Edge]

_第一次报告是在2021年11月23日。_
