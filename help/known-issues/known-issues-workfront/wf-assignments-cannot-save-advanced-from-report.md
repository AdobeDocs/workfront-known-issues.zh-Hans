# 分配：无法从报表中保存高级分配更改

当用户尝试从报表中进行高级分配更改时，更改不会保存，并且用户会看到以下消息：

“422： APIModel INTERNAL不支持字段removeOldAssignments”

**解决方法**

您可以从项目的任务列表中更改高级分配。
