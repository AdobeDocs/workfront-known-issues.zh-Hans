# 任务：前置任务工具提示不显示任务名称

>[!NOTE]
>
>此问题已于2024年3月28日修复。

当用户查看任务列表并将鼠标悬停在前置任务图标上以获取更多信息时，显示的工具提示不显示前置任务的名称。
