# 自定义表单：无法添加“estimateByHours”或“hoursPerPoint”任务字段

当用户尝试将“estimateByHours”或“hoursPerPoint”字段添加到表单生成器中的自定义表单时，显示错误消息：“这是一个无效的自定义表达式，请重试。”

_首次报告于2024年10月9日。_
