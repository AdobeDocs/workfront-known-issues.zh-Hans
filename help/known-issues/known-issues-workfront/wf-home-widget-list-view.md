# 主页：用户移动主页小组件中的列时创建的&quot;widget.list.view&quot;

>[!NOTE]
>
>此问题已于2024年8月22日修复。

当管理员在设置区域查看视图列表时，他们会看到许多称为“widget.list.view”的视图。 这些视图是在用户重新排列主页小部件中的列时创建的。
