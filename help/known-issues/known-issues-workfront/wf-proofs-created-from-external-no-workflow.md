# 验证：从链接文档创建的验证未应用验证模板

<!--On WF, WFF, WFP TOCs-->

>[!NOTE]
>
>此问题已于 2024 年 3 月 14 日修复。

当用户从链接文档创建验证时，未正确应用验证模板，并且验证可能缺少工作流等信息。

这一点还适用于通过 API 和通过 Workfront Fusion 创建的验证。

_首次报告于 2024 年 2 月 28 日。_
