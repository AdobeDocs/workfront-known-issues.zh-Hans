# 校样：从链接文档创建的校样未应用校样模板

<!--On WF, WFF, WFP TOCs-->

当用户从链接文档创建校样时，未正确应用校样模板，并且校样可能缺少工作流等信息。

这一点还适用于通过 API 和通过 Workfront Fusion 创建的校样。

_首次报告于 2024 年 2 月 28 日。_
