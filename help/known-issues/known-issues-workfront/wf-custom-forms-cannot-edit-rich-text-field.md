# 自定义表单：无法编辑富文本字段

>[!NOTE]
>
>此问题已于2025年12月22日修复。

当用户尝试在自定义表单上编辑富文本字段时，更改不会保存，并且字段不会更新以反映更改。

_首次报告于2025年12月3日。_
