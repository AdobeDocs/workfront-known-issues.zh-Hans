# 用户：创建用户时较低的访问级别不可用

>[!NOTE]
>
>此问题已于2024年3月14日修复。

当一个用户在创建另一个用户时，只有第一个用户的访问级别对新创建的用户可用。权限低于创建用户的那些用户的所有访问级别都应可分配给新用户。

_首次报告于 2024 年 1 月 30 日。_
