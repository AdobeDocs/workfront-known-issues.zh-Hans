# [!UICONTROL 更新]：无法在[!UICONTROL 更新]区域记录当天的时间

当用户尝试在[!UICONTROL 更新]区域中记录当天的时间时，当前日期将灰显，并且用户无法输入任何小时。

**解决方法**

清除浏览器的缓存或使用无痕窗口可以暂时解决此问题。

_首次报告于 2023 年 3 月 7 日。_
