# [!UICONTROL 更新]：无法记录中当天的时间 [!UICONTROL 更新] 区域

当用户尝试在中记录当天的时间时 [!UICONTROL 更新] 区域，当天显示为灰色，用户无法输入任何小时。

**解决方法**

清除浏览器的缓存或使用无痕窗口可以暂时解决此问题。
