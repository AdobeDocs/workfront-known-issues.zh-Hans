# 验证：验证仍保留在“我的审批”小组件上

>[!NOTE]
>
>此问题已于2024年3月28日修复。

应该从“我的审批”构件中消失的验证仍保留在该构件中。 当多个用户同时做出验证决策，或者一个用户做出决策并快速更改它时，可能会发生这种情况。

_首次报告于2024年2月7日。_
