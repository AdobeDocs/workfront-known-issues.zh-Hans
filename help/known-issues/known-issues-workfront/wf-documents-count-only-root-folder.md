# 文件：文件数量不准确

>[!NOTE]
>
>此问题已于2024年7月25日修复。

当用户查看带有附加文档的对象时，左侧导航中的“文档”选项卡可能会显示不准确的文档数量。显示的数量是根文件夹中的文档数。子文件夹中的任何文档均不计算在内。

_首次报告于 2024 年 6 月 20 日。_
