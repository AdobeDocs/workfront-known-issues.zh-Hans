# 模板：无法从中选择模板 [!UICONTROL 收藏模板]

当用户尝试从“操作”（三个圆点）菜单中选择模板时，当用户将其鼠标移动到该列表时，模板列表会消失，并且用户无法选择模板。 这是因为滚动条位于菜单和模板列表之间，并且鼠标在滚动条移动到模板列表时会聚焦在滚动条上。

**解决方法**

尝试下列操作之一：

* 将鼠标移动到收藏夹列表，而不触摸滚动条。
* 从“[!UICONTROL 从模板转换为项目]“ ”选项。
