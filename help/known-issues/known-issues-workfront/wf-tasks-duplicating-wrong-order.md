# 任务：创建或重复任务导致任务列表以错误的顺序显示

当用户在任务列表上创建或重复任务时，任务列表随后根据任务顺序编号以错误的顺序显示这些任务。任务编号正确，但其显示顺序有误。

**变通方法**

刷新页面。
