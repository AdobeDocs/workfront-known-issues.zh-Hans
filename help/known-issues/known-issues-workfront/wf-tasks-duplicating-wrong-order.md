# 任务：创建或复制任务导致任务列表显示顺序错误

当用户创建或复制任务列表上的任务时，任务列表会根据任务顺序编号以错误的顺序显示任务。 任务编号正确，但其显示顺序有误。

**变通方法**

刷新页面。
