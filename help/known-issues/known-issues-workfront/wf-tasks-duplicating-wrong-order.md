# 任务：复制任务导致任务按错误顺序显示

当用户在一个任务列表中选择多个任务并复制它们时，任务列表将根据任务顺序编号以错误的顺序显示任务。 任务编号正确，但显示顺序错误。

**解决方法**

刷新页面。
