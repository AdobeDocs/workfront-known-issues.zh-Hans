# [!DNL Workfront]：未应用品牌化

当用户将文件上传到设置的[!UICONTROL 品牌化]区域时，似乎正确上传了这些文件。但如果用户查看[!UICONTROL 主菜单]，却看不到徽标，而是看到以下消息：

&quot;[!UICONTROL 公司徽标不可用。我们未能根据所提供的 URL 加载您公司的徽标。请联系您的系统管理员。]&quot;
