# 更新：通过 API 发表的评论在沙盒环境中不可见

>[!NOTE]
>
>此问题已于 2024 年 4 月 29 日修复。

当通过 Workfront API 发表评论时，该评论在生产环境中可见。但是，该评论在沙盒环境中不可见。

_首次报告于 2024 年 4 月 8 日。_
