# 用户：批量更新用户非常慢

>[!NOTE]
>
>此文章已于2025年1月21日修复。

当用户尝试批量编辑用户时，编辑窗口需要很长时间才能打开。即使只有两个用户也是如此，用户越多花费的时间就越长。
