# 项目：批量编辑项目时出错

当用户尝试批量编辑项目时，编辑窗口为空白，并且用户会看到以下错误：

“发生意外错误”

用户必须刷新页面才能返回项目列表或报告。

从项目列表或项目报告进行批量编辑时已报告会出现此问题。

**解决方法**

刷新页面，然后尝试再次编辑。
