# 文档：无法上传到 AEM 文件夹文档列表

>[!NOTE]
>
>此问题已关闭，因为无法复制。

用户无法将文件拖放到 AEM 文件夹内的主文档列表区域。

**解决方法**

可以将文件放入 AEM 文件夹（文档列表区域之外）进行上传。
