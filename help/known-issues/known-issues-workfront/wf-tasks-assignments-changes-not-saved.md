# 任务：在任务的“编辑”对话框中进行的编辑未保存

>[!NOTE]
>
>* 此问题适用于旧版任务体验。
>* 产品团队目前正在评估此问题的解决方案，这可能需要产品增强功能。 产品增强功能在“产品公告”中而非“维护更新”中传送。

当用户在旧版任务对话框中更改任务的编辑部分并保存时，更改未保存。

**解决方法**

对于某些更改，您可以使用新的任务体验。
