# 任务：对任务的“工作总揽”部分的编辑未保存

>[!NOTE]
>
>此问题适用于旧版任务体验。

当用户在旧版任务对话框中更改任务的编辑部分并保存时，更改未保存。

**解决方法**

对于某些更改，您可以使用新的任务体验。
