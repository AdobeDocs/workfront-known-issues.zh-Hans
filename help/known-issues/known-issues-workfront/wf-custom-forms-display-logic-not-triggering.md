# 自定义表单：显示逻辑在请求表单上无法正确显示

>[!NOTE]
>
>此问题已于2026年2月13日修复。

当用户填写连接到请求的自定义表单时，显示逻辑无法遵循，并且某些字段可能无法按预期显示。

**解决方法**

关闭请求草稿，然后重新打开它。
