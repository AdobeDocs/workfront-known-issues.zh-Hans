# 设置：无法以其他用户身份登录

>[!NOTE]
>
>* 此问题已于2025年Ju；3日修复。
>* 此问题已在“预览Sandbox”环境中报告。

当Workfront管理员尝试使用“登录身份”功能以其他用户身份登录时，他们不会以其他用户身份登录，并且会看到以下消息：

“糟糕！出现问题。请联系 Workfront，以便我们找出错误并加以修复。”

_首次报告于2025年6月26日。_
