# 展示板：Workfront 原生字段不显示在卡片上

>[!NOTE]
>
>此问题已于2024年5月2日修复。

当用户查看配置为显示本机（非自定义）字段的卡片时，这些字段在卡片上不可见。

**解决方法**

单击卡片可以转到其代表的对象。
