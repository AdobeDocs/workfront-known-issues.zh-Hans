# [!UICONTROL 资源管理]：无法更改日期 [!UICONTROL 利用] 区域

当用户查看 [!UICONTROL 利用] 面积 [!UICONTROL 资源管理] 和尝试更改日期范围，日期不会更改，并且用户会看到以下错误消息：

“[!UICONTROL 出现错误，我们正在寻找解决问题的方法。要继续您的工作，请尝试再次刷新此浏览器。]&quot;
