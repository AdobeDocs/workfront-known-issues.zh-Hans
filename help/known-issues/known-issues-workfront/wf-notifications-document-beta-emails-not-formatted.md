# 通知：文档审批测试版的通知格式不正确

当用户收到来自新文档审批测试版的电子邮件时，该电子邮件的格式不正确。

Microsoft Outlook 中已针对 Windows 报告了此问题。

_首次报告于 2024 年 3 月 14 日。_
