# 集成： [!DNL SharePoint] 文件夹所有者未显示

>[!NOTE]
>
>此问题已关闭，因为它不是 [!DNL Workfront] 错误。

当用户尝试查看新 [!DNL SharePoint] [!UICONTROL （图形API）] 集成时，文件夹不会显示。 即使用户是文件夹的所有者，也会发生这种情况。

_第一次报告是在2022年8月19日。_
