# 主页：无法删除 [!UICONTROL 待办事项] 新建中的项目 [!UICONTROL 主页] 体验

当用户在新 [!UICONTROL 主页] experience尝试从删除项目 [!UICONTROL 待办事项] 小部件，该项目未删除，且用户看到以下错误消息：

&quot;[!UICONTROL 移除您的待办事项时出现问题，请稍后再试。]&quot;

当有小时记录到时，可能会发生这种情况 [!UICONTROL 待办事项] 项目。

_首次报告于 2023 年 7 月 28 日。_
