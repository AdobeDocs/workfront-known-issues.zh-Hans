# 请求：无法访问请求页面(“……/submitted” URL)上的仪表板

>[!NOTE]
>
>此问题已于2024年5月16日修复。

当试图访问请求页面上的仪表板时，URL会暂时显示仪表板URL，然后快速切换到已提交的URL (/submitted)。

_首次报告于2024年5月16日。_
