# 时间表： [!UICONTROL 小时类型] 下拉列表未打开

当用户尝试打开 [!UICONTROL 小时类型] 时间表上的下拉列表，屏幕闪烁，但下拉列表未打开。 当监视器设置为异常分辨率，或者浏览器窗口的缩放未设置为100%时，会发生这种情况。

**解决方法**

请尝试以下操作之一：

* 确保浏览器窗口上的缩放比例设置为100%
* 调整显示器分辨率

_首次报告于 2023 年 1 月 30 日。_
