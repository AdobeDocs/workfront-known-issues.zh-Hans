# 敏捷： [!UICONTROL 看板] 编辑卡片时展示板留空

用户在 [!UICONTROL 看板] 然后点击离开卡片， [!UICONTROL 看板] 展示板留空，而不是使用编辑进行刷新。 如果用户手动刷新页面，则 [!UICONTROL 看板] 展示板返回，显示正确的编辑。

**解决方法**

刷新页面。
