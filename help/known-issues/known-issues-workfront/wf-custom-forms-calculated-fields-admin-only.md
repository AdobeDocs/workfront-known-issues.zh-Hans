# 自定义表单：计算字段未显示非管理员的数据

>[!NOTE]
>
>此问题已于2026年2月19日修复。

当不是Workfront管理员的用户查看自定义表单时，其字段从表单的仅管理员可用分区中提取数据，则该字段不显示计算值。 如果用户运行“重新计算表达式”，则会显示值，但刷新时该值会丢失。

Workfront管理员可以按预期查看字段值。

_首次报告于2026年1月13日。_
