# 项目：无法向已完成的项目添加问题

>[!NOTE]
>
>此问题已于2024年10月24日修复。

当用户尝试将问题添加到处于完成状态的项目时，用户无法添加问题。 即使启用了“添加和编辑问题”选项，也可能发生这种情况。

_首次报告于2024年10月14日。_
