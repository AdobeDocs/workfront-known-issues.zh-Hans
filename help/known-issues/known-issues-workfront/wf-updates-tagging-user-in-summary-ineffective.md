# 更新：在“摘要”面板中使用@标记用户无效

当用户尝试使用“摘要”面板的“更新”区域中的@标记另一个用户时，单击下拉菜单中的用户名无效。 再次尝试标记用户如预期般工作。

**解决方法**

再次标记用户。 第二次尝试按预期进行。

_首次报告于 2024 年 3 月 18 日。_
