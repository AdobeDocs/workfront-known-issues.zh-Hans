# 更新：在“摘要”面板中使用 @ 标记用户无效

当用户尝试在“摘要”面板的“更新”区域中使用 @ 标记另一个用户时，单击下拉菜单中的用户姓名无效。第二次尝试标记用户时，效果与预期一样。

**解决方法**

再次标记用户。第二次尝试，效果与预期一样。

或

使用 Enter/Return 键选择用户。

_首次报告于 2024 年 3 月 18 日。_
