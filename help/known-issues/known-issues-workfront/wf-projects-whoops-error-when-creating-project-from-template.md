# 项目：从模板创建项目时出现“[!UICONTROL 糟糕]”错误

当用户尝试从模板创建项目时，项目未创建，并且用户会看到以下错误消息：

“[!UICONTROL 糟糕！ 出现问题。 请联系 [!DNL Workfront]，以便我们找出错误并加以修复。]”

**解决方法**

创建一个空白项目，然后将模板附加到该项目。
