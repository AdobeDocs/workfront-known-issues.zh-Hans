# 更新：标记用户无效

当用户在评论中被标记时，该评论对标记的用户不可见，此外，无论通过电子邮件还是应用程序内通知，标记的用户都不会收到有关评论的通知。

该问题已在旧版评论体验中报告。

_首次报告于2024年3月1日。_
