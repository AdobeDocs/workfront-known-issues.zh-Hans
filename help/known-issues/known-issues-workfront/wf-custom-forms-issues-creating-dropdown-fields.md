# 自定义表单：自定义下拉字段中的问题

启用为多选字段的自定义下拉字段可能会显示以下问题：

* “+”[!UICONTROL 添加]“ ”按钮在表单未处于编辑模式时不存在。
* 没有值的字段显示“ — [!UICONTROL 无标签] — ”选项。

**解决方法**

尝试下列操作之一：

* 删除要多选的字段的选项。
* 设置默认值。
