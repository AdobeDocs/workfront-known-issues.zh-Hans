# 任务和问题：创建任务或问题时意外退出

>[!NOTE]
>
> 已在2022年9月8日为任务修复此问题。

当用户在项目中创建任务或问题并在创建弹出窗口之外单击时，弹出窗口将关闭且不发出警告，并且之前输入的所有信息都将丢失。

_第一次报告是在2022年3月25日。_
