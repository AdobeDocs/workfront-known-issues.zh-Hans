# 任务/问题：已分配用户未获得对对象的访问权限

当用户被分配给工作对象时，用户可能无法获得对该对象的访问权限，并且无法完成分配。 用户可以请求访问权限，但这增加了完成工作项的时间和复杂性。

**解决方法**

执行下列操作之一：

* 取消分配并重新分配用户
* 手动授予用户访问权限
