# 资源规划：[!UICONTROL 更多]按钮不可见

当用户查看资源规划程序并尝试单击[!UICONTROL 更多]按钮时，按钮不可见。

**解决方法**

单击[!UICONTROL 更多]按钮所在的区域。按钮不可见，但可以点击。
