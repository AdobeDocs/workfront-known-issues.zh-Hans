# 用户：提醒用户注册时显示的错误消息不正确

当用户提醒另一个用户注册时，提醒电子邮件按预期发送，但发送提醒的用户看到错误消息&quot;[!UICONTROL 我们无法发送邀请电子邮件。 您可以稍后重新发送或联系您的 [!DNL Workfront] 管理员。]”这可能会导致混淆是否发送了电子邮件。
