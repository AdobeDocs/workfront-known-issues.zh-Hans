# 任务：创建任务时无法选择用户角色

>[!NOTE]
>
>此问题已于2025年12月21日修复。

当用户创建任务并尝试分配用户角色时，分配角色的选项不存在，用户无法分配用户角色。

**解决方法**

创建任务，然后编辑任务以在“高级工作”中分配用户角色。
