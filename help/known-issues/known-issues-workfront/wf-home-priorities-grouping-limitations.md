# 优先级：分组阻止排序和内联编辑

>[!NOTE]
>
>产品团队目前正在评估此问题的解决方案，这可能需要产品增强功能。 产品增强功能在“产品公告”中而非“维护更新”中传送。

当优先级工作列表应用了分组时，无法对其进行排序，并且内联编辑无法用于工作列表中的字段。

_首次报告于2024年10月3日。_
