# 项目：在产品列表上使用过滤器时出错

如果用户查看项目列表并应用过滤器，则该列表会消失并且用户会看到以下消息：

&quot;[!UICONTROL 出现错误，我们正在努力解决该请求。 要继续您的工作，请尝试刷新此浏览器页面。]”

使用新的过滤器体验时已报告此问题。

_首次报告于2024年2月22日。_
