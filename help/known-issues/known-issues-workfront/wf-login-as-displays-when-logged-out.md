# 登录身份：在注销“登录身份”时，主页和pin显示不准确

当以其他用户身份登录的用户以该用户身份注销时，他们会看到以下内容：

* 主页显示他们作为用户登录的用户名
* 他们的针不存在

这些用户只会影响显示。 该用户实际上已作为其他用户注销。

**解决方法**

刷新页面。
