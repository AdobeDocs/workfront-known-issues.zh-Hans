# 主页：项目名称显示“无访问权限”

<!--valid issue, won't fix until legacy home is deprecated-->

当用户在主页上查看“我的任务”小部件时，项目名称列中不会显示”无访问权限“。即使项目名称在“我的工作”小部件中可见，也会发生这种情况。
