# 文档：文档区域未显示所有文档

当用户查看对象的“文档”区域时，并非所有附加到文档的文档都显示。

此问题仅适用于文档区域的标准视图。 “列表”视图按预期运行。

**解决方法**

切换到文档区域的列表视图。
