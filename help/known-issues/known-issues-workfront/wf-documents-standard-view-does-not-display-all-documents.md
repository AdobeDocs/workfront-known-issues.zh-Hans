# 文档：文档区域不显示所有文档

当用户查看对象的文档区域时，并不显示附加到该文档的所有文档。

此问题仅适用于文档区域的标准视图。列表视图功能按预期运行。

**解决方法**

切换到文档区域的列表视图。
