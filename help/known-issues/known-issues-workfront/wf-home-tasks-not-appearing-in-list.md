# [!UICONTROL 主页]：项目未出现在[!UICONTROL 主页工作列表中]

当用户查看[!UICONTROL 主页工作列表]时，应该出现在[!UICONTROL 工作列表]中的项目不存在。例如，用户可能会转到某个项目，看到他们在项目中被分配了任务，但这些任务不会出现在用户的[!UICONTROL 主页工作列表]中。
