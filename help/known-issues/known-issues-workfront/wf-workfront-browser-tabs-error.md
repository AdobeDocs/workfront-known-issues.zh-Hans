# Workfront：保持打开的浏览器选项卡加载失败

当用户的Workfront浏览器选项卡保持足够长的打开时间，以便Workfront需要再次登录，而不是提示用户再次登录时，页面显示错误：“500-001a：无法加载页面”。

**解决方法**

关闭注销的浏览器选项卡并再次打开Workfront将正确地提示您登录Workfront。
