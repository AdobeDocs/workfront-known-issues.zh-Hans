# Workfront：打开的浏览器选项卡无法加载

当用户长时间打开 Workfront 浏览器选项卡，以至于 Workfront 要求再次登录时，页面不会提示用户再次登录，而是显示错误：“500-001a：无法加载页面”。

**解决方法**

关闭已注销的浏览器选项卡并再次打开 Workfront 将正确提示您登录 Workfront。
