# 仪表板：用户无法访问 [!UICONTROL 画布功能板]

当一个用户应该有权访问 [!UICONTROL 画布功能板] 尝试访问 [!UICONTROL 画布功能板]，仪表板未加载，且用户看到以下消息：

&quot;[!UICONTROL 您无权访问画布功能板，请联系管理员以获取访问权限。]&quot;

在以下位置禁用以下所有访问级别选项时，会发生这种情况 [!UICONTROL 报表]：

* [!UICONTROL 删除]
* [!UICONTROL 公开共享报告]
* [!UICONTROL 在系统范围内共享]

_首次报告于 2023 年 10 月 25 日。_
