# 文档：使用文档的公共链接时出错

<!--
>[!NOTE]
>
>This issue has been resolved.-->

当用户单击公共链接转到 Workfront 中的文档时，他们可能会看到以下错误：“该文件已不存在”。

**解决方法**

如果用户已登录，他们可以按预期查看文档。
