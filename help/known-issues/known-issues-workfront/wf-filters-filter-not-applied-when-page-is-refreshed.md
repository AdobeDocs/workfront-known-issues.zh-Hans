# 过滤器：刷新页面时不应用过滤器

当用户查看应用了过滤器的列表并刷新页面时，刷新的页面会像未应用过滤器一样显示结果。

已在以下方面报告了此问题：

* 项目

**解决方法**

删除并重新应用过滤器。
