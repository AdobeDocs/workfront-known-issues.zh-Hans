# 更新：某些用户看不到附加到注释的图像或文档

当用户查看对象的更新，并且更新附加了图像或文档时，图像或文档未显示在用户的更新中。

已报告非Workfront管理员用户出现此问题。

_首次报告于2025年3月3日。_
