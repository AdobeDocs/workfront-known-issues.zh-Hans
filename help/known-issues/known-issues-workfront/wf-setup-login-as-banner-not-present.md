# 设置：以其他用户身份登录时，登录横幅不会出现。

当管理员以其他用户身份登录时，不会出现“登录身份”横幅。用户不会收到以其他用户身份登录的提醒，也无法退出此用户帐户。

**解决方法**

刷新页面。
