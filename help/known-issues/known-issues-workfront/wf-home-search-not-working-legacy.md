# [!UICONTROL 主页]：搜索功能在旧版中不起作用 [!UICONTROL 主页工作列表]

>[!NOTE]
>
>此问题已得到解决。

当用户查看其 [!UICONTROL 主页工作列表] 在旧版主页体验中，如果尝试搜索列表，则搜索无效。

_首次报告于 2023 年 8 月 18 日。_
