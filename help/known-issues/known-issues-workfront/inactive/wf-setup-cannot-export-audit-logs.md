# [!UICONTROL 设置]：无法导出审核日志

>[!NOTE]
>
>产品团队目前正在评估此问题的解决方案。 解决此问题后，将在“产品公告”中进行沟通，而非在“维护更新”中进行沟通。

当用户尝试导出大量审核日志时，日志不会导出。没有迹象表明导出失败。

**解决方法**

在一次导出中导出较少的日志。
