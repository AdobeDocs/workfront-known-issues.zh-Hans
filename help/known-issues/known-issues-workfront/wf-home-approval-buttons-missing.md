# 主页：等待审批小组件中缺少审批按钮

<!--
>[!NOTE]
>
>This issue was fixed on February 13, 2025.
-->

在主页中使用等待审批小组件时，不会出现选择审批决策的按钮。

**解决方法**

可以通过直接导航至审批来做出审批决策。
