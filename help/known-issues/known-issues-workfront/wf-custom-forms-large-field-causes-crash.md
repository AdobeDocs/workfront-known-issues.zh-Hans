# 自定义表单：无法编辑字段列表页面上的字段

当用户尝试在字段列表页面上编辑大字段时，页面崩溃并出现以下错误：

“[!UICONTROL Aww Snap，显示此页面时出现问题。 错误代码5]

已在基于Chromium的浏览器中报告此问题

**解决方法**

尝试下列操作之一：

* 直接在表单中编辑字段
* 使用其他浏览器，如Firefox或Safari
