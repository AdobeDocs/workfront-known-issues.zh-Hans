# 任务： [!UICONTROL 计划小时数] 当将附加用户分配给任务时重新分配

当用户被分派到具有 [!UICONTROL 计划小时数] 分配至任务的其他被分配人，任务的 [!UICONTROL 计划小时数] 均等分配给任务的所有被分配人。
