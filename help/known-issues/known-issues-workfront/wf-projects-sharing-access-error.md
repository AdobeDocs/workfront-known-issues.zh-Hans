# 项目：由于“访问删除”错误，无法共享项目

尝试共享项目时，用户收到错误：“您没有足够的权限删除此项目”。 尽管用户不尝试删除项目，并且用户是项目所有者并且拥有项目的共享和管理访问权限，但是仍然会发生这种情况。
