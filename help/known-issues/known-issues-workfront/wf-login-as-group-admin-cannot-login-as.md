# 登录身份：组管理员无法以其他用户身份登录

当组管理员尝试以其他用户身份登录时，他们不会以用户身份登录。 相反，“登录身份”横幅将显示组管理员的名称，组管理员将看到他们自己的帐户。

_首次报告于2025年2月18日。_
