# 用户：创建用户时无法使用高级选项

>[!NOTE]
>
>此问题已于2025年10月27日修复。

当Workfront管理员创建用户并尝试使用高级选项时，他们无法使用选项，并且会看到以下消息：

```
Cannot read properties of undefined (reading 'sections')

Cannot read properties of undefined (reading 'sections')

Error fetching object details
```

_首次报告于2025年9月29日。_
