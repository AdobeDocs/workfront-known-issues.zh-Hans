# 更新：导航离开并返回评论时未保留文本

当用户编写包含@mention的评论，然后在提交评论之前导航离开并返回评论时，@mention注草稿中缺少在评论之后输入的任何文本。

已在新的评论体验中报告了此问题。

_首次报告于2024年3月21日。_
