# 项目：使用模板创建的项目与模板的优先级不匹配

>[!NOTE]
>
>此问题已于2024年5月7日解决。

当用户使用模板将请求转化为项目时，新创建项目的优先级与模板或请求不匹配。

_首次报告于2024年5月2日。_
