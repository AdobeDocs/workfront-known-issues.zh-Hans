# 敏捷：无法编辑信息卡上自定义表单的字段

>[!NOTE]
>
>产品团队当前正在评估此问题解决方案，这可能需要产品增强。 产品增强功能在产品公告中传递，而不是与维护更新一起传递。

当看板板上卡片上的字段从自定义表单中提取时，用户无法直接从卡片编辑该字段。

**解决方法**

打开任务并编辑 [!UICONTROL 编辑任务] 窗口。

_第一次报告是在2022年4月12日。_
