# 状态：隐藏状态时出现[!UICONTROL ‘哎呦’]错误

当用户试图隐藏状态时，该状态不会被隐藏，并且用户会看到以下错误：

“[!UICONTROL 糟糕！ 出现问题。请联系 Workfront，以便我们找出错误并加以修复。”]&quot;
