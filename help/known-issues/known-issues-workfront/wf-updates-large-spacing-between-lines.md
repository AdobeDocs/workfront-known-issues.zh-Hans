# 更新：行与行之间有大空格

当用户使用Return或Enter键键入包含多行的更新，或将多行粘贴到更新中时，更新会按预期显示。 但是，如果用户随后在报表中查看该更新，则行之间会有很大的空格。

已在新的评论体验中报告了此问题。

_首次报告于2024年1月19日。_
