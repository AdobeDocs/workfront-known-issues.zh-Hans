# 时间表：未正确生成时间表

当用户尝试生成时间表，并且生成时间设置为“每隔一周”时，仅生成一周。

**解决方法**

通过设置>系统>诊断>生成时间表来手动生成时间表。

_首次报告于2025年6月2日。_
