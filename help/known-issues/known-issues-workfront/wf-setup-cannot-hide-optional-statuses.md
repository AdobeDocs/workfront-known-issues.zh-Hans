# [!UICONTROL 设置]：无法隐藏可选状态

>[!NOTE]
>
>此问题已于2024年1月18日修复

当用户尝试隐藏系统和组级别的可选状态时，状态未隐藏。 如果用户查看状态，则隐藏状态的选项不会启用，即使用户启用了并保存了更改。

_首次报告于1月10日。_
