# 工作负载均衡器：未正确计算每个用户的小时数

>[!NOTE]
>
>此问题已于2026年2月6日修复。

当用户在工作负载均衡器查看用户的小时数时，这些小时数可能无法正确计算。 例如，周视图可能显示一种小时数计算方法，但4周视图显示该周的不同小时数。

_首次报告于2026年1月27日。_
