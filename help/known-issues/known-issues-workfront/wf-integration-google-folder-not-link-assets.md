# 集成：从 [!DNL Google Drive] 链接文件夹不会链接文件夹中的资产

当用户将 [!DNL Google Drive] 中的文件夹链接到 Workfront 时，该文件夹中的资产无法链接。文件夹中的资产数量显示为 0。

**解决方法**

重新验证您的Google连接。
