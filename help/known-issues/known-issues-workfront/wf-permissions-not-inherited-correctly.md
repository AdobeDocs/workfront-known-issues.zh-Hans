# 权限：对象权限继承不正确

>[!NOTE]
>
>产品团队目前正在评估此问题的解决方案，这可能需要产品增强功能。 产品增强功能在“产品公告”中而非“维护更新”中传送。

继承的权限未正确应用于对象。 发生此情况可能是因为继承的权限比较复杂，这可能受以下影响：

* 该对象与大量人员共享
* 大量对象受继承的权限更改影响

**解决方法**

限制对象的大小或复杂性有助于避免此问题。 我们建议您在任意父对象下拥有的子对象不超过10,000个。
