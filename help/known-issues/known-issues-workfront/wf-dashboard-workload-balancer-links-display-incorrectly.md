# 功能板：工作负载平衡器链接在功能板中显示不正确

>[!NOTE]
>
>此问题已于2022年8月25日修复。

将可共享的工作负载平衡器链接作为外部页面添加到功能板时，该链接显示不正确。 功能板不使用与链接关联的唯一视图/过滤器，而是使用最近应用的视图/过滤器来平衡工作负载。

_第一次报告是在2022年8月10日。_
