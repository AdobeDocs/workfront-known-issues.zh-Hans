# 验证：无法在预览环境中创建验证

>[!NOTE]
>
>此问题仅在“预览Sandbox”环境中报告。

当用户尝试在预览沙盒环境中创建验证时，验证未创建，并且用户看到“无法生成”消息。

_首次报告于2024年10月17日。_
