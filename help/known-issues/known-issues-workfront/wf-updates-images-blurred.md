# 更新：更新中的图像预览很模糊

<!--

>[!NOTE]
>
>This issue was fixed on April 11, 2024.

-->

当用户查看更新并单击图像上的放大镜来预览图像时，打开的预览非常模糊。

如果用户下载该图像，则该图像会以预期的分辨率显示。
