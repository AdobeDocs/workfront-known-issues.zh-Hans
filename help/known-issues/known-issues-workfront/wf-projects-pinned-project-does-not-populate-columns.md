# 项目：固定项目未在某些列中显示信息

当用户使用pin导航到固定项目时，对象列表（如任务列表）可能会显示空白列。 例如， [!UICONTROL 指定任务] 列可能不会显示任何分配，即使已进行了分配。

**解决方法**

重新加载页面。
