# 项目：在某些列中，已固定的项目不显示信息

当用户使用大头针导航到已固定的项目时，对象列表（例如任务列表）可能会显示空白列。例如，[!UICONTROL 任务]列中可能显示没有任何任务，即使已经分配了任务。

**解决方法**

重新加载页面。
