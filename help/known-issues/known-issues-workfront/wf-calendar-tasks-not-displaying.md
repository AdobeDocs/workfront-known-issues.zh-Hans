# 日历：任务未显示

>[!NOTE]
>
>产品团队当前正在评估此问题解决方案。 此问题解决后，将在产品公告中传递，而不是与维护更新一起传递。

当用户具有列表报表和具有相同过滤器的日历报表时，他们会看到某些任务显示在列表报表中，但不显示在日历报表中。
