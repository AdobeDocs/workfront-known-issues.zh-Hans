# 更新：团队名称未显示在@mention

当新 [!DNL Adobe Workfront] 体验会尝试在更新中标记团队，并且团队名称包含多个词语（例如“团队名称”），系统不会通知这些词语，也不会显示在“收件人”字段中。 如果在 [!UICONTROL 通知] 字段，系统会按预期通知他们，但更新中只会对部分名称添加超链接。

_第一次报告是在2021年6月3日。_
