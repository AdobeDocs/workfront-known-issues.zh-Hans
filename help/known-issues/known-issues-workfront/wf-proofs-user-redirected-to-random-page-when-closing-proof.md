# 校样：关闭校样时，用户不会返回到上一页

>[!NOTE]
>
>此问题已于2022年9月29日修复。

<!--This is on the WF page as well as the WFP page-->

在中查看校样的用户 [!DNL Workfront] 关闭校样，他们不会返回在打开校样之前所在的页面。 而是会将用户重定向到 [!DNL Workfront].

_第一次报告是在2022年9月15日。_
