# 自定义表单：字段显示逻辑未正确遵循

当用户填写包含显示逻辑（其中某些字段是否基于其他字段中的值显示）的自定义表单时，这些字段未以遵循该字段逻辑的方式显示。

_首次报告于2025年10月6日。_
