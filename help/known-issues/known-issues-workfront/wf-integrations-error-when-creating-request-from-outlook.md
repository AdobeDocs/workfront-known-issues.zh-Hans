# 集成：从提交请求时出错 [!DNL Outlook]

当用户尝试提交请求时， [!DNL Outlook] 电子邮件中，一个或多个附件未上传，用户看到以下错误：

&quot;[!UICONTROL 发生以下错误：具有句柄xxxx的文件不存在。]&quot;

仅当通过请求队列或在创建请求时手动为新请求分配时，才会发生这种情况。

_第一次报告是在2022年8月10日。_
