# 设置：无法编辑工作角色的“成本/小时”或“计费/小时”



>[!NOTE]
>
>产品团队目前正在评估此问题的解决方案，这可能需要产品增强功能。 产品增强功能在“产品公告”中而非“维护更新”中传送。

当用户编辑工作角色并尝试编辑“成本/小时”或“计费/小时”字段时，这些字段为空，用户无法编辑它们。

**解决方法**

编辑工作角色列表中的内联字段。
