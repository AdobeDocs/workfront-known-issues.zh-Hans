# 请求：无法打开请求列表的页面超出第一页

>[!NOTE]
>
> 此问题已关闭，因为它可按设计运行。

当用户查看的请求列表包含2000个以上的请求（因此是多个页面），并尝试查看该列表的另一页时，“[!UICONTROL 页面]“ ”按钮无响应，用户无法查看列表的其他页面。

_第一次报告是在2022年7月28日。_
