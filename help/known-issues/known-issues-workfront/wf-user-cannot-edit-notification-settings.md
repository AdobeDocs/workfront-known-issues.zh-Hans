# 用户：用户无法编辑自己的通知设置

当用户 [!UICONTROL 工作人员] 许可证会尝试编辑自己的通知设置， [!UICONTROL 通知] 选项在 [!UICONTROL 编辑] ，则用户无法编辑设置。
