# [!UICONTROL 主页]： [!UICONTROL 请求] 构件不显示所有已分配的团队

当用户查看 [!UICONTROL 请求] 新版本中的构件 [!UICONTROL 主页] experience中，即使请求实际分配了多个团队，也显示为仅分配了一个团队。
