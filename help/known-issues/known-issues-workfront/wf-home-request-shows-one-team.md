# [!UICONTROL 主页]：[!UICONTROL 请求]构件不显示所有分配的团队

当用户在新的[!UICONTROL 主页]体验中查看[!UICONTROL 请求]构件时，请求显示为仅被分配了一个团队，即使实际上已将多个团队分配给这些请求也是如此。
