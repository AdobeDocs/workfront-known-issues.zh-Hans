# [!UICONTROL 主页]：[!UICONTROL 请求]构件未显示所有分配的团队

>[!NOTE]
>
>产品团队目前正在评估此问题的解决方案，这可能需要产品增强功能。 产品增强功能在“产品公告”中而非“维护更新”中传送。

当用户在新的[!UICONTROL 主页]体验中查看[!UICONTROL 请求]构件时，请求显示为仅分配了一个团队，即使实际上为请求分配了多个团队也是如此。
