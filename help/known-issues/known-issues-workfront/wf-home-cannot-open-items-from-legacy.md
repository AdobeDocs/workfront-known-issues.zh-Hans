# 主页：无法打开以下位置的项目： [!UICONTROL 主页日程表] 或 [!UICONTROL 工作列表]

当用户尝试从他们的打开工作项或验证时 [!UICONTROL 主页日程表] 或 [!UICONTROL 主页工作列表]，则不会打开该项目。

**解决方法**

刷新页面。
