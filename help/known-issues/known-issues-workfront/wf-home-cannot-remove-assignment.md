# [!UICONTROL 主页]:无法删除 [!UICONTROL 主页]

当用户在 [!UICONTROL 家庭工作列表] 并尝试删除该分配，页面将刷新，且该分配尚未删除。

**解决方法**

打开问题，并从问题本身中删除分配。
