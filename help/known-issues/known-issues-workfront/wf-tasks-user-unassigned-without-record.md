# 任务：一个或多个任务中已取消分配用户

系统可能会自动从用户分配到的任务中取消分配用户。 这可能发生在一个或多个任务中。 取消分配不会显示在任务的系统更新区域中，尽管它确实显示在设置菜单的更新馈送区域中。

_首次报告于2023年12月6日。_
