# 任务和问题：无法编辑任务或问题的名称或完成百分比

具有 [!UICONTROL Contribute] 访问任务或问题时，无法在标题中编辑任务或问题的名称。 此外，具有 [!UICONTROL Contribute] 访问无法编辑任务或问题的完成百分比。

_第一次报告是在2022年10月6日。_
