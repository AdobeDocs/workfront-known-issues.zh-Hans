# 用户：重新激活用户时未分配访问级别

当用户重新激活已停用的用户，并尝试在 [!UICONTROL 重新激活用户] 窗口中，访问级别下拉列表不会按用户类型填充，并且用户无法选择访问级别。 如果用户在访问级别中键入并保存，则该访问级别不会分配给重新激活的用户。

**解决方法**

重新激活用户，然后在 [!UICONTROL 编辑人员] 窗口。
