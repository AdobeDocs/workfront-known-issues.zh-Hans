# 布局模板：已删除的视图仍然可用

当管理员通过“布局模板”从列表中删除视图时，该视图仍会显示在用户可用视图的列表中。 选中后，视图将不应用或应用视图的列，而不提供任何数据。

_首次报告于2025年1月21日。_
