# 设置：以其他用户身份注销时出现问题

当管理员以其他用户身份登录并尝试注销时，可能会出现以下问题：

* 用户将被重定向到登录页面，而不是他们自己的帐户
* 用户看到错误：“[!UICONTROL 您无权访问Workfront。]”

_首次报告于2024年7月30日。_
