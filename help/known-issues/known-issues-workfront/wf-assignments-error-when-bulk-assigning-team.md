# 分配：批量分配团队时出错

当用户在分配个人后批量编辑任务或问题并分配团队时，不会保存分配，并且用户会看到以下错误：

&quot;[!UICONTROL 我们再试一次 — 出现以下错误：teamAssignments必须是对象列表或ID列表]&quot;

**解决方法**

先分配团队，然后是个人。

_第一次报告是在2022年10月7日。_
