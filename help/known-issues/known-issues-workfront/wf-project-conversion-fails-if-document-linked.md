# 项目：如果附加了 Web 验证，则无法将问题转换为项目

>[!NOTE]
>
>此问题已于2024年4月18日修复。

当问题附加了 Web 验证（使用来自外部文档提供商（例如 SharePoint）的链接的 URL 验证），并且用户尝试将该问题转化为项目时，转化会失败并且不会创建项目。用户会看到以下错误：

“复制文件（文件 GUID）时出现问题。请删除该文件或联系支持人员并重试。”

_首次报告于 2024 年 3 月 4 日。_
