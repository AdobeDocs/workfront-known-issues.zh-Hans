# 项目：如果附加Web验证，则无法将问题转化为项目

当问题附加了Web验证(使用SharePoint等外部文档提供商的链接的URL验证)，并且用户尝试将该问题转化为项目时，转化失败并且未创建项目。 用户看到以下错误：

“复制文件（文件GUID）时出现问题。 请移除文件或联系支持部门并重试。”

_首次报告于 2024 年 3 月 4 日。_
