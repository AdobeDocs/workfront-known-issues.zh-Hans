# 项目：如果附加了文件，则无法将问题转换为项目

<!--

>[!NOTE]
>
>This issue was fixed on May 23, 2024.

-->

当问题有文档并当用户尝试将该问题转换为项目时，转换会失败并且不会创建项目。用户会看到以下错误：

“复制文件（文件 GUID）时出现问题。请删除该文件或联系支持人员并重试。”

来自外部提供商的文件和本地文件均已报告此问题。

_首次报告于 2024 年 3 月 4 日。_
