# 项目：如果附加文档，则无法将问题转化为项目

<!--

>[!NOTE]
>
>This issue was fixed on April 18, 2024.

-->

当问题具有文档，并且用户尝试将该问题转化为项目时，转化失败并且未创建项目。 用户会看到以下错误：

“复制文件（文件 GUID）时出现问题。请删除该文件或联系支持人员并重试。”

外部提供程序文件和本地文件均已报告此问题。

_首次报告于 2024 年 3 月 4 日。_
