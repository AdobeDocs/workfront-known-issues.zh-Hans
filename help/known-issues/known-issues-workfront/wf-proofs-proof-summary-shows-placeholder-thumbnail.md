# 验证：[!UICONTROL 打印摘要]不显示图像缩略图

<!--This is on both the WF and WFP TOCs-->

>[!NOTE]
>
>此问题已关闭，因为它就是如此设计的。

当用户在 [!DNL Workfront] 中查看验证的[!UICONTROL 打印摘要]页面时，不显示该验证的缩略图。而是出现一个简单的占位符图像。
