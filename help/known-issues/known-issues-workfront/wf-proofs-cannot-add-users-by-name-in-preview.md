# 验证：无法在预览环境中按名称将用户添加到验证

>[!NOTE]
>
>此问题已在“预览Sandbox”环境中报告

当用户尝试将另一个用户添加到验证并按名称搜索该用户时，该名称未出现，并且用户无法选择该名称以将另一个用户添加到验证中。

即使在刷新预览沙盒环境后，也可能发生这种情况。

_首次报告于2025年5月5日。_
