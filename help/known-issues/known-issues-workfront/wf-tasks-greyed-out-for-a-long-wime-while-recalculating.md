# 任务：重新计算时间线时，任务日期变灰很久

当用户在任务列表中作出触发重新计算时间线的更改时，还将重新计算该时间线影响的其他任务。当发生这种情况时，这些其他任务中受影响的日期变灰。在此问题中，这些任务保持变灰很久。

将保存在这些日期变灰时对这些日期作出的更改。

**解决方法**

刷新页面。
