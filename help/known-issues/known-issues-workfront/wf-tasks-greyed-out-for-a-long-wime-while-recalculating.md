# 任务：重新计算时间线时，任务日期因延长时间而灰显

当用户在触发重新计算时间线的任务列表中进行更改时，也会重新计算受该时间线影响的其他任务。 在此过程中，这些其他任务中受影响的日期将呈灰显状态。 在此问题中，任务会长时间保持灰显状态。

将保存对这些灰显日期所做的更改。

**解决方法**

刷新页面。
