# 自定义表单：通过列编辑自定义表单时出错

当用户查看包含自定义表单链接的对象列表并尝试通过单击链接来编辑自定义表单时，用户无法编辑表单，而是会看到以下消息：

“出现了错误，我们正在积极努力解决它。”

**解决方法**

打开自定义表单所附加到的对象，并通过该对象的详细信息页面编辑自定义表单。
