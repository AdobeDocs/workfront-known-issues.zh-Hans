# 搜索：高级搜索未返回预期结果

>[!NOTE]
>
>此问题已于2025年6月2日修复。

当用户使用高级搜索时，不会返回预期结果。 即使没有将过滤器应用于高级搜索，也可能发生这种情况。

**解决方法**

基本搜索按预期运行。
