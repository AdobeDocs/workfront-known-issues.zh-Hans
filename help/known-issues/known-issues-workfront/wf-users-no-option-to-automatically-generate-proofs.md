# 用户：用户配置文件中缺少自动生成验证的选项

>[!NOTE]
>
>此问题已于2025年12月17日修复。

当用户查看用户配置文件时，“上传文档时自动生成验证”选项不可用。 即使满足该选项的所有先决条件，也可能发生这种情况。

_首次报告于2025年12月5日。_
