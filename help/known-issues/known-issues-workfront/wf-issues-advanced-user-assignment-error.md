# 请求和问题：高级用户分配错误

当用户尝试对请求或问题进行高级用户分配时，用户未被分配，并且会看到以下错误：

“出现错误，我们正在寻找解决问题的方法。要继续工作，请尝试刷新此浏览器页面。”

刷新浏览器页面无效。
