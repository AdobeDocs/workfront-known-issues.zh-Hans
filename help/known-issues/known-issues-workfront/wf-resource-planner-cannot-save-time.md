# [!UICONTROL 资源计划员]:添加时间后无法保存

当 [!UICONTROL 资源计划员] 添加时间，然后尝试保存， [!UICONTROL 保存] 按钮无响应。 如果用户单击 [!UICONTROL 取消]，则他们会看到以下消息：

&quot;[!UICONTROL 您有未保存的更改！如果您离开此页面，则更改将丢失。是否继续？]&quot;

**解决方法**

选择默认过滤器。 当 [!UICONTROL 确认] 出现消息，单击 [!UICONTROL 保存并继续].
