# 自定义表单：构建大型表单时浏览器崩溃

>[!NOTE]
>
>此文章已于2024年10月3日修复。

当用户在表单设计器中构建或编辑大型表单时，浏览器可能会崩溃，导致用户无法创建或编辑该表单。

这会影响表单设计器。该表单可以按照预期用于对象上。
