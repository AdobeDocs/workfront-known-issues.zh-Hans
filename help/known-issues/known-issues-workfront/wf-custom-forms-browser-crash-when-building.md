# 自定义表单：构建大型表单时浏览器崩溃

当用户在表单设计器中生成或编辑大型表单时，浏览器可能会崩溃，并且用户无法创建或编辑表单。

这会影响表单设计器。 可以按预期在对象上使用表单。
