# 验证：审批选项未出现在 [!UICONTROL 校对查看器]

当用户尝试在 [!UICONTROL 桌面校对查看器]，在请求审批后不久，审批选项不可见，用户无法审批验证。 审批选项会在请求审批数小时后显示。

**解决方法**

批准文档 [!UICONTROL 文档详细信息] 页面位置 [!DNL Workfront].
