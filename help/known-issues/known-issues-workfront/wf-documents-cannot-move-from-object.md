# 文档：无法从对象的“文档”选项卡移动文档

>[!NOTE]
>
>此文章已于2024年10月10日修复。

当尝试从项目、任务或问题的文档选项卡移动文档时，用户遇到以下错误：“加载辅助内容时出错”。

**解决方法**

通过直接导航到“文档详细信息”页面，单击三个点的“文档操作”菜单，然后单击“移动”，仍然可以移动文档。

_首次报告于2024年10月3日。_
