# 验证：创建新验证版本会删除验证的所有版本

<!--WF and WFP TOCs-->

当用户从文档创建验证时，将创建验证。 但是，如果用户创建另一个版本的验证，则旧版本和新版本都会被删除。 有一个 [!UICONTROL 创建校对] 选项，并且验证版本可以在 [!UICONTROL 垃圾桶] 区域 [!UICONTROL 校对] 区域位于 [!DNL Workfront].
