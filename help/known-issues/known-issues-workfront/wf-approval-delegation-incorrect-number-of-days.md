# 审批：为不正确的天数设置了审批委派

>[!NOTE]
>
>此问题已关闭，因为它不构成问题。

当用户安排”个人休息时间“并委派响应审批时，审批委派可能包括安排的休息时间之前或之后的天数。

**解决方法**

这种差异是由于用户档案中的时区与用户分配的时间表的时区之间的差异造成的。

我们建议为用户工作的每个时区创建一个唯一的时间表，并将每个用户分配到与其用户档案中的时区匹配的时间表。
