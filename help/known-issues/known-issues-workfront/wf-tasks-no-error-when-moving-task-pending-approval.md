# 任务：在中对任务重新排序时没有错误消息 [!UICONTROL 待批准] 状态

当用户尝试对任务列表中的任务重新排序，并且该任务位于 [!UICONTROL 待批准] 状态时，任务会在任务列表中移动。 刷新后，用户会看到项目未移动。 项目无法移动，因为它位于 [!UICONTROL 待批准] 状态，但没有消息告知用户项目无法移动，这可能会导致混淆。

_首次报告于 2023 年 4 月 12 日。_
