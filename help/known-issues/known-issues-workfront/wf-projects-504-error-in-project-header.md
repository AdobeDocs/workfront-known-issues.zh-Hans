# 项目：项目标题中的 504 错误

>[!NOTE]
>
>此问题已得到解决。

当用户查看项目时，项目标题中没有名称或数据。相反，用户会看到类似于以下内容的错误：

“504：未定义的 /internal/qs/customizableHeaderData?objCode=PROJ&amp;objectID=xxxxxxxxxxxxxxx”

_首次报告于 2023 年 2 月 3 日。_
