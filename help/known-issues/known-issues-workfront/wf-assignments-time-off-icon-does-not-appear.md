# 任务：分配任务时不显示休息时间图标

当用户分配任务并开始输入在执行任务期间安排休息时间的受让人的姓名时，“建议任务”区域下的用户图标不会显示休息时间（飞机）图标。如果用户继续输入，以使名称在“用户和团队”区域下显示，则会显示该图标。

**解决方法**

继续输入名称，直到它在“用户和团队”区域出现。
