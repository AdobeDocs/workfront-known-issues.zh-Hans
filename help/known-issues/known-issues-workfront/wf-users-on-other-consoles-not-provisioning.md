# 用户：用户未在 Adobe Admin Console 中正确预配

>[!NOTE]
>
>此问题已于2026年2月3日修复。

如果某个用户已存在于 Adobe Admin Console 中，随后又自动预配到包含 Workfront 的控制台，则该用户不会正确预配到新的控制台中。

_首次报告时间：2025 年 11 月 20 日。_
