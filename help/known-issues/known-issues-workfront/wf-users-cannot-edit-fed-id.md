# 用户：无法编辑用户的联合ID

>[!NOTE]
>
>此问题会影响尚未迁移到Adobe Identity Management系统的组织

当管理员尝试更新用户的联合ID时，此信息的字段不可用，管理员无法更新它。

这意味着用户无法登录到Workfront。

_首次报告于2025年9月9日。_
