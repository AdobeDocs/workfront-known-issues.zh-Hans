# 自定义Forms:将问题转换为项目时，不会保留按部分时间隐藏的字段中的数据

>[!NOTE]
>
> 此问题已于2022年8月11日修复。

当用户将问题转换为项目，并且问题包括一个自定义表单，该表单中的某个节点有数据（可以使用显示逻辑隐藏），则该节中的数据不会传递到新项目。

_第一次报告是在2022年7月20日。_
