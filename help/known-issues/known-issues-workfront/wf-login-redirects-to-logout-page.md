# 登录：登录页面重定向到注销登陆页面

当用户尝试登录 [!DNL Workfront]，而不是登录页面，而是被定向到他们要登出时会登陆的页面。

**解决方法**

输入您的 [!DNL Workfront] URL并尝试再次登录。
