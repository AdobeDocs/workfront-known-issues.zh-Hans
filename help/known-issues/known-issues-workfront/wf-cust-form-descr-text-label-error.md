# 自定义表单：由于“描述性文本”字段，无法编辑自定义表单

>[!NOTE]
>
> 此问题已于2022年8月11日修复。

当用户尝试编辑包含 [!UICONTROL 描述性文本] 字段中，字段的标签将不会填充。 用户看到错误“[!UICONTROL 此字段为必填字段]“ ”下，且用户由于出现此错误而无法编辑自定义表单。

_第一次报告是在2022年7月14日。_
