# 验证：验证状态不会在[!UICONTROL 文档]列表中更新

当验证状态发生变化时，显示该验证的[!UICONTROL 文档]列表不会显示正确的新状态。

[!DNL Workfront] 中的[!UICONTROL 验证工作流]页面显示验证的正确状态。
