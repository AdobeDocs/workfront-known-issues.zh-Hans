# 验证：无法创建新版本的验证

当用户尝试创建新版本的验证时，不会创建新版本，并且用户会看到以下错误消息：

&quot;[!UICONTROL 生成校对时出现错误]&quot;

**解决方法**

创建新验证而不是新验证版本。
