# 工时单：向时间表添加注释时出错

当用户在添加工时时尝试向工时单添加评论时，会保存时间，但不会保存评论。 用户还会看到以下错误消息：

* [!UICONTROL 您无权为其他用户记录小时数].
* [!UICONTROL 您最近的更改未保存。刷新页面以查看上次保存的更改].

_第一次报告是在2022年9月27日。_
