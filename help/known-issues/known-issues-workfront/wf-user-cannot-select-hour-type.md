# 用户：无法选择[!UICONTROL 默认小时数类型]或[!UICONTROL 可用小时数类型]

当管理员正在编辑用户并尝试选择[!UICONTROL 默认小时数类型]或[!UICONTROL 可用小时数类型]时，他们会看到这些字段的下拉列表已禁用，并且他们无法选择小时数类型。

_首次报告于 2023 年 3 月 30 日。_
