# 用户：无法编辑其他用户的通知设置

>[!NOTE]
>
>此问题已于2025年9月25日修复。

当用户尝试编辑其他用户的通知设置时，无法编辑设置，并且会看到以下消息：

“您没有足够的权限编辑财务数据。”

_首次报告于2025年9月9日。_
