# 更新：标记用户然后导航离开会删除已标记的用户

>[!NOTE]
>
>此问题已于2022年8月4日修复。

当用户键入另一个用户以在更新流中通知他们，然后他们导航到另一个选项卡并返回到更新流时，将删除已标记的用户。

_第一次报告是在2022年7月7日。_
