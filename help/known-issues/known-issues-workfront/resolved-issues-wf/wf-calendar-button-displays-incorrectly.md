# 日历：将项目添加到日历时，按钮显示不正确

>[!NOTE]
>
>此问题已于2022年7月25日修复。

当用户将项目添加到日历并选择 [!UICONTROL 添加高级项目] 选项， [!UICONTROL 添加任务], [!UICONTROL 添加项目]或 [!UICONTROL 添加元素] 按钮与文本重叠。 按钮仍可正常使用，用户可以添加项目。

_第一次报告是在2022年6月2日。_
