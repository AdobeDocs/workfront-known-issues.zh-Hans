# 设置：“登录身份”导致出现空白屏幕

>[!NOTE]
>
>此问题已于2024年8月8日修复。

当管理员以其他用户身份登录时，不会看到该用户的帐户，而是会看到一个空白屏幕。

**解决方法**

刷新屏幕。 这可能需要多次尝试。

_首次报告于2024年7月17日。_
