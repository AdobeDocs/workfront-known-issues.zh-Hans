# 任务/问题：在任务或问题的展示板视图中添加工作项时出现问题

>[!NOTE]
>
>此问题已于2023年12月7日修复。

当用户查看项目中的任务或问题区域的展示板视图并尝试添加任务或问题时，可能会出现以下情况：

* 弹出窗口在两种不同的窗口样式之间来回切换
* 用户无法关闭弹出窗口

根据报告，当用户在输入任何信息之前从弹出窗口的左侧导航中选择一个区域时，将出现此情况。

_首次报告于 2023 年 11 月 23 日。_
