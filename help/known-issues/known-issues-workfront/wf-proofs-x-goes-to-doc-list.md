# 验证：关闭验证重定向到文档列表

当用户查看验证并单击X图标将其关闭时，用户被重定向到文档列表而不是文档详细信息。

当以下两项均成立时，会发生这种情况：

* 用户已从链接打开验证。 如果从文档详情打开验证，则在关闭验证时将用户定向回文档详情。
* 用户属于尚未迁移到Adobe Unified Shell的组织。
