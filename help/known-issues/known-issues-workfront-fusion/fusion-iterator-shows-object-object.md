# [!DNL Workfront Fusion]：[!UICONTROL 迭代器]模块中的数组元素显示为[!UICONTROL 对象 Object]

当用户查看[!UICONTROL 迭代器]模块时，数组元素显示为[!UICONTROL 对象 Object]。即使之前查看时数组元素已正确显示，也可能发生此情况。
