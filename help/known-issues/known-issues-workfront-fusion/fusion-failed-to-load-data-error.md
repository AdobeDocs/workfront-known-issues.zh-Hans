# [!DNL Workfront Fusion]:&quot;[!UICONTROL 无法加载数据]“错误”

[!DNL Workfront Fusion] 与 [!DNL Workfront] 帐户可能无法进行身份验证，从而导致情景引发以下错误：

[!UICONTROL 无法加载数据！ 由于上一个请求失败，请求失败。]

在恢复连接之前，无法执行该方案。

**解决方法**

创建新连接，然后在 [!DNL Workfront] 模块。

_第一次报告是在2022年9月15日。_
