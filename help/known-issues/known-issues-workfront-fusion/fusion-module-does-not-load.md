# [!DNL Workfront Fusion]:创建或编辑方案时，模块未加载

当用户创建或编辑方案并尝试打开模块时，该模块不会打开。 用户看到一个旋转的圆圈，表示模块正在加载，但模块未完成加载。

_第一次报告是在2022年9月12日。_
