# Workfront Fusion：从现在开始，开始设置“从现在开始”处理所有数据

选择“从现在开始”作为方案的“选择何时开始”设置后，如果用户将方案还原到以前的版本，则它将尝试处理所有数据，而不管日期如何，而不是仅包括选择“开始”选项后的数据。
