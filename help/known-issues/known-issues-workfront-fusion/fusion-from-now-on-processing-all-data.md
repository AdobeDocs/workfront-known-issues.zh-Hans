# Workfront Fusion：开始设置“从现在开始”处理所有数据

在选择&quot;从现在开始&quot;作为场景的&quot;选择何时开始&quot;设置后，如果用户将场景恢复到以前的版本，它将尝试处理所有数据而不管日期如何，而不是仅包含选择开始选项后的数据。
