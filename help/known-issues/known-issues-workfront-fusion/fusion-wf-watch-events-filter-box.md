# [!DNL Workfront Fusion]：无法在中输入值 [!DNL Workfront] > [!UICONTROL 观看活动] 筛选

当用户配置 [!DNL Workfront] > [!DNL Watch Events] 模块中，并且运算符设置为&quot;[!UICONTROL 等于]，”用于输入值的文本框不存在。

**解决方法**

选择其他运算符，然后选择“[!UICONTROL 等于]”再次。
