# [!DNL Workfront Fusion]：无法在 [!DNL Workfront] > [!UICONTROL 监测活动]筛选条件中输入值

当用户配置 [!DNL Workfront] > [!DNL Watch Events] 模块并将运算符设置为[!UICONTROL 等于]时，不出现用于输入值的文本框。

**解决方法**

选择其他运算符，然后再次选择“[!UICONTROL 等于]”。
