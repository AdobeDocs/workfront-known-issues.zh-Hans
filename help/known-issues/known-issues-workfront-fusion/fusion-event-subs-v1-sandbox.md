# Workfront Fusion：连接到沙盒环境时，Workfront事件订阅版本2选项不可用

>[!NOTE]
>
>此问题已于2025年5月5日修复。

当用户配置用于监视Workfront沙盒环境中的事件的Workfront模块时，基于Workfront事件订阅服务版本2的标准不可用。

_首次报告于2025年3月10日。_
