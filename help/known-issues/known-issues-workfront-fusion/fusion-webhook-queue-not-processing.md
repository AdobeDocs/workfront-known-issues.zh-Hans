# Workfront Fusion：Webhook 队列未处理

>[!NOTE]
>
>此问题已于2024年9月13日修复。

当发生一个应该触发 Workfront Fusion 中的 webhook 场景的事件时，该场景不会执行，事件将保留在 webhook 的队列中。
