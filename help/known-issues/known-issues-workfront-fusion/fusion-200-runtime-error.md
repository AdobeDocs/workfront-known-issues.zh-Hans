# Workfront Fusion：来自Workfront模块的200响应出现RuntimeError

Workfront模块可以返回 `RuntimeError [200]` 响应。 200表示响应成功，但错误显示请求不成功。

如果响应时间非常长，则可能会发生这种情况。 数据会返回到Fusion，但Fusion无法处理该数据。
