# [!DNL Workfront Fusion]:401错误 [!DNL Workfront] 模块


>[!NOTE]
>
>此问题已于2022年6月30日修复。

各种 [!DNL Workfront] 模块返回 [!UICONTROL 401 InvalidAccessTokenError] 中。 此错误是间歇性的，在同一模块中成功操作后可能会发生。

**解决方法：**

+ 重新授权在模块中使用的连接，可临时恢复功能。

**第一次报告是在2022年4月22日。**
