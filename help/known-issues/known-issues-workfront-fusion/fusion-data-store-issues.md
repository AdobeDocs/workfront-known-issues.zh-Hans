# Workfront Fusion：数据存储问题

关于数据存储中的布尔字段，已报告以下问题：

* 数据存储不会返回任何具有 EMPTY 值的字段的值。
* 用户不能在数据存储中直接将值设置为 FALSE。

**解决方法**

要将值设置为 FALSE，请使用适当的更新记录模块。
