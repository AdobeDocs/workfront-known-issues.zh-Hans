# 更新：不显示附加到系统更新的评论

<!--
>[!NOTE]
>
>This issue has been closed because it is working as designed.
-->

>[!NOTE]
>
>此问题已关闭，因为其正在按设计的那样发挥作用。

当用户对系统操作（如状态更改）发表评论时，不在“更新”区域中显示该评论。

已在新的评论体验中报告了此问题。
