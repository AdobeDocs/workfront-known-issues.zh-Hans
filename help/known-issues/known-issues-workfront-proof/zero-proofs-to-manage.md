# [!DNL Workfront Proof]:功能板在中显示0个校样 [!UICONTROL 要管理的校样] 和 [!UICONTROL 等待决策的证明] 报告

>[!NOTE]
>
>此问题已于2022年8月4日修复。

当用户查看 [!DNL Workfront Proof] 功能板， [!UICONTROL 要管理的校样] 和 [!UICONTROL 等待决策的证明] 报表会按不同类别（总计、准时等）显示0个报表。 如果用户单击其中任何类别，则他们会看到该类别中应显示在功能板上的校样列表。

第一次报告是在2022年5月23日。
