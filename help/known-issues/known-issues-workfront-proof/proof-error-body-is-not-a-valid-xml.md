# [!DNL Workfront Proof] API：“[!UICONTROL 正文不是有效的 XML]”错误

<!--On WFP and WFF TOCs-->

>[!NOTE]
>
>由于无法重现，此问题已关闭。

对 [!DNL Workfront Proof] API 的调用返回以下错误：

“[!UICONTROL 正文不是有效的 XML。根标记未闭合]”

当直接访问 [!DNL Workfront Proof] API 或通过 [!DNL Workfront Fusion] 访问它时，可能会发生此情况。
