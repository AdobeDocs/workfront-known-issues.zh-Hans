# [!DNL Workfront Proof]:每日摘要电子邮件中不包含“[!UICONTROL 查看详细信息]“ ”按钮

>[!NOTE]
>
>此问题已关闭，因为它不是问题。

当用户收到 [!DNL Workfront Proof] 每日摘要电子邮件中，电子邮件中不包含“[!UICONTROL 查看详细信息]“ ”按钮。 受此行为影响的校样没有引用ID。

_第一次报告是在2022年6月10日。_
