# 校样：在 [!UICONTROL 校样查看器] 实时不可见

>[!NOTE]
>
>22.3版本已解决此问题。

用户在中查看校样时 [!UICONTROL 校样查看器]，而另一用户则发表评论，则不会立即向查看用户显示评论。 查看用户必须刷新 [!UICONTROL 校样查看器] 手动查看注释。

第一次报告是在2021年12月3日。
