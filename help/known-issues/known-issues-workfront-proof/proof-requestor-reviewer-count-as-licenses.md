# [!DNL Workfront Proof]:请求者和审阅者计入组织的许可证计数

当用户作为审阅者或请求者添加到校样时，他们会获得“[!UICONTROL 访客]“权限配置文件”，不应使用 [!DNL Workfront Proof] 许可证。 但是，添加用户时，使用的计数 [!DNL Workfront Proof] 许可证数量增加。

_第一次报告是在2022年6月23日。_
