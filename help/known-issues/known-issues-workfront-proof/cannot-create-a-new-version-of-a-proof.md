# 校样：无法创建校样的新版本

>[!NOTE]
>
>此问题已于2022年3月29日修复。

当用户尝试创建校样的新版本时，不会生成新校样，用户会看到以下消息：

```
Error
Error generating proof
```

**解决方法**:

使用将为新版本的文件创建新校样（而非校样的新版本），然后对链接到新校样的旧版本进行评论。

第一次报告是在2022年2月15日。
