# [!DNL Workfront Proof]:查看包含多个页面的校样时显示空白页面

>[!NOTE]
>
>此问题已于2022年9月29日修复。

当用户查看包含多个页面的校样时，用户可以查看页面的缩略图，但页面不会在主查看器中打开。

_第一次报告是在2022年9月22日。_
