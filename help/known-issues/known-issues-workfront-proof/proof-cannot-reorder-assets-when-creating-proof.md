# [!UICONTROL Workfront Proof]：创建验证时无法重新排序资源

>[!NOTE]
>
>此问题已关闭。

当用户在 [!UICONTROL Workfront Proof] 尝试重新排序将合并到验证中的资产，他们无法拖动文件，文件的顺序未更改。

为验证对资产重新排序在中按预期工作 [!DNL Workfront].

_首次报告于2024年1月4日。_
