# [!DNL Workfront Proof]:在中的初始配置后，无法修改用户选项 [!UICONTROL 工作流模板]

>[!NOTE]
>
>本文已于2022年8月18日修复。

用户将用户添加到 [!UICONTROL 工作流模板]，则可以配置选项。 但是，初始配置完成后，用户将无法再修改以下内容：

* &quot;[!UICONTROL 解决注释并应用操作]“能力”
* &quot;[!UICONTROL 通过标记共享校样]“能力”
* 校样角色([!UICONTROL 审阅人], [!UICONTROL 审批者]等)

**第一次报告是在2022年5月31日。**
